== Description ==
<h3>Ключевые особенности плагина интернет-магазина WP-Shop:</h3>
<ul>
	<li>Товары строятся на основе обычных записей WordPress, к которым, при указании цены, добавляется ценник и кнопка «купить»</li>
	<li>Есть удобные инструменты для создания «витрин» товаров и прайс-листов, с различными настройками</li>
	<li>Оффлайновые сценарии оплаты и доставки: Самовывоз из магазина/склада/офиса; Наличными курьеру, Оплата при получении на почте (наложенный платеж)</li>
	<li>Настраиваемые варианты доставки: можно сконструировать любые варианты доставки вместо стандартныx</li>
	<li>Онлайн-сценарии оплаты: Webmoney(напрямую), Все виды электронных денег + карты VISA и MASTERCARD, через платежные системы РОБОКАССА и ЕДИНАЯ КАССА</li>
	<li>Интеграция с платежной системой Hronopay (Хронопей) - (в разработке)</li>
	<li>Личный кабинет покупателя: просмотр сделанных покупателем заказов, хранение истории покупок, профиля покупателя</li>
	<li>Продажа «цифровых товаров»: загрузка файлов после их оплаты, с защитой от несанкционированного скачивания</li>
	<li>Реализована дисконтная система, которая формирует скидку от суммы заказа по формуле, заданной администратором магазина</li>
	<li>Генерация Яндекс.XML файла для трансляции товаров в Яндекс.Маркет и другие подобные сервисы</li>
	<li>Система «массового» управления товарами: быстрое изменение цен, актуализация товарных остатков, снятие с продажи товаров, которых временно нет в наличии</li>
	<li>Редактор шаблонов писем клиентам</li>
	<li>Локализация текстов и подписей: позволяет перевести с русского и адаптировать магазин под любую языковую среду</li>
	<li>Оплата картами VISA и MASTERCARD через PayPal</li>
	<li>Интегрирована статистика Google Analytics и Яндекс.Метрика для интернет-магазинов: считает продажи, средний чек, другую специфичную статистику для интернет-магазинов.</li>
	<li>Импорт товаров и синхронизация данных с 1С-Бухгалтерией (в разработке)</li>
</ul>
и многое другое

== Installation ==

1. Upload plugin "WP Shop" to the `/wp-content/plugins/` directory
2. Upload plugin "cForms 14.5" (http://wp-shop.ru/files/cforms_145.zip) to the `/wp-content/plugins/` directory
3. Activate the plugin cForms 14.5 through the 'Plugins' menu in WordPress
4. Activate the plugin WP Shop through the 'Plugins' menu in WordPress
5. See [full userguide](http://www.youtube.com/embed/Ew51hmmwVIg?list=PLSDnqQFsL1p38epx0edCjawemN-g5_8Ql) how to set up your shop
